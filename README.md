# Açık Kaynak Paket Yöneticisi Türkçe(OSPMT)

**OSPMT bir komut üreteci ve düzenleyicisidir.** Paketleri seçmenize yardımcı olur ve bunları kurmak, güncellemek veya kaldırmak için ihtiyacınız olan terminal komutlarını oluşturur. Sisteminize **DOKUNMAZ**.

## Nasıl Çalışır (Katı Akış)
1.  **Paket Yöneticisi Seç**: APT, DNF, Pacman, AUR (yay), Flatpak, Winget, Chocolatey, Homebrew arasından seçim yapın.
2.  **Paket Seç**: İstediğiniz yazılımı arayın ve seçin.
3.  **İşlem Seç**: Yükle, Güncelle veya Kaldır.
4.  **Kod Üret**: Komutu görmek için butona tıklayın.
5.  **Manuel Uygulama**: Komutu kopyalayın ve kendi terminalinizde çalıştırın.

## Desteklenen Yöneticiler
*   **APT**: Debian/Ubuntu
*   **DNF**: Fedora
*   **Pacman**: Arch Linux
*   **AUR (yay)**: Arch Kullanıcı Deposu
*   **Flatpak**: Evrensel
*   **Winget**: Windows
*   **Chocolatey**: Windows
*   **Homebrew**: macOS

## Temel Prensipler
*   **Sıfır Otomasyon**: Bu araç sizin yerinize asla bir komut çalıştırmaz.
*   **Sıfır Sihir**: Gizli bağımlılıklar yok, arka plan betikleri yok.
*   **Tam Şeffaflık**: Neyin çalıştırılacağını tam olarak görürsünüz.

## Yasal Uyarı
Bu araç sisteminizi değiştirmez. Kopyalayıp yapıştırdığınız komutlardan siz sorumlusunuz.
